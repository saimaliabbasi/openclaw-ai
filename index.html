⚙️ Setup — API Keys

OpenClaw needs 2 free API keys to work. Follow the steps below:

1
Groq API Key FREE
Used for AI (Llama 3 model). No credit card needed.
→ Get free key at console.groq.com
GROQ_API_KEY=gsk_xxxxxxxx...
2
Tavily Search Key FREE
Used for web search. 1,000 searches/month free.
→ Get free key at tavily.com
TAVILY_API_KEY=tvly-xxxxxxxx...
3
Gmail App Password OPTIONAL
Only needed for sending emails. Skip if not using email.
① Enable 2-Step Verification on your Google Account
② Go to: Google Account → Security → App Passwords
③ Create an app password → copy the 16-character code
GMAIL_USER=you@gmail.com
GMAIL_APP_PASSWORD=xxxx xxxx xxxx xxxx
Paste all keys into: openclaw/.env — then restart the server.
